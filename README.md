# CLI Dictionary 

An offline command line interface dictionary with over **176,000 WORDS!** 

## Usage

``` cli-dict <word> ```

## Dependencies 

Not sure how portable this is,

* ruby 3.2
* ruby gem sqlite3

Dont forget to add it to your PATH
